# Tinkercad

## Circuito sugerido
Simulacao de leitura de temperatura/umidade (DHT) e luminosidade (LDR), com acionamento de LED e buzzer.

## Componentes
- Microcontrolador compativel no Tinkercad.
- Sensor DHT (ou sensor equivalente de simulacao).
- LDR + resistor.
- LED + resistor.
- Buzzer.
- Protoboard e jumpers.

## Evidencias necessarias
- Print do circuito montado.
- Print da simulacao em execucao.

## Limitacoes da simulacao
O Tinkercad nao simula Wi-Fi/MQTT. A validacao de comunicacao deve ser demonstrada no hardware fisico ou em simulacao separada.
