# Roadmap do Semestre - CanTrace

| Periodo | Entregas principais |
| --- | --- |
| Maio | Fundacao do repositorio, documentacao inicial e definicao de escopo. |
| Maio - Junho | Modulo IoT (simulacao, topicos MQTT e dashboard Node-RED). |
| Junho | Requisitos e especificacao academica. |
| Julho | Modelagem (UML, ER, arquitetura). |
| Agosto - Setembro | Backend (planejamento e inicio de implementacao). |
| Setembro - Outubro | Frontend (planejamento e inicio de implementacao). |
| Outubro - Novembro | Testes e ajustes de qualidade. |
| Novembro - Dezembro | Documentacao final e preparacao da banca. |

Observacoes: o cronograma e sujeito a ajustes conforme disponibilidade de hardware, definicao de requisitos e alinhamento com a disciplina.
